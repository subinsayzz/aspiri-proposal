# ASPIRI Brand Launch Proposal

This proposal deck is built with React, Vite, Tailwind CSS, and Framer Motion.
It follows the design language of the Gymrupt proposal but adapted for the ASPIRI jewelry brand with a Gold/Luxurious aesthetic.

## Features
- **Design**: Glassmorphic, dark premium theme with Gold accents.
- **Content**: 90-Day Fractional CMO Engagement Pilot.
- **Launch Ideas**: Includes "The Golden Circle", "Heirloom Stories", and "Virtual Atelier".
- **Integration**: Showcases the "Gymrupt Growth Engine" (App) as the infrastructure for the launch.

## How to Run

1.  Navigate to the directory:
    ```bash
    cd aspiri-proposal
    ```

2.  Install dependencies (if not already installed):
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open the link in your browser (usually `http://localhost:5173`).

## Navigation
- Use **Left/Right Arrow Keys** to navigate slides.
- Click arrows on screen.
